项目描述：基于人脸识别的嵌入式考勤系统
本项目参考B站开源视频教程（BV1xw4m1R7Jr），成功复刻并移植到阿尔法开发板（ARM平台），实现了一套完整的人脸识别考勤系统。系统采用客户端/服务器架构，客户端运行于嵌入式设备，通过摄像头采集人脸图像，利用OpenCV进行人脸检测与识别，完成员工签到/签退；服务器端基于Qt开发，提供考勤记录管理、用户信息维护、考勤数据统计与导出等功能。

主要功能
人脸注册：录入员工姓名、工号及面部特征模板
实时考勤：识别摄像头前的人脸，自动记录签到时间与状态
考勤管理：查看、筛选、导出考勤记录（支持Excel/CSV）
网络通信：客户端与服务器通过TCP/IP同步数据，支持多设备同时上报
嵌入式优化：针对ARM开发板进行交叉编译与性能调优，降低资源占用

技术栈
嵌入式平台：阿尔法开发板（ARM Cortex-A系列）
软件开发：Qt 5.12 + OpenCV 4.5.2(pc端运行) / OpenCV 3.4.1(arm端/i.mx6ull开发板上运行)
人脸识别：基于Haar Cascade或LBPH特征（可替换为深度学习模型）
数据库：SQLite（本地存储） + MySQL（服务器可选）
通信协议：自定义TCP数据包，JSON格式交互

适配与移植说明
本项目已完全脱离x86环境，在阿尔法开发板上运行稳定。所有依赖库（Qt、OpenCV）均通过交叉编译链构建，并对图像采集、识别算法进行了帧率优化，确保嵌入式端实时性（识别延迟<200ms）。代码结构清晰，注释完整，便于二次开发或集成到其他ARM Linux设备中。
